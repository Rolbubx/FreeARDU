# FreeARDU - Bare-Metal Cortex-M7 Project

This project is a bare-metal firmware for Cortex-M7 (i.MX RT1060) featuring a custom startup, OS-style runtime, and graphical framebuffer.

## Features

- Custom Bare-Metal Startup (`startup.S`)
- Linker script for i.MX RT1060
- Framebuffer support with automatic screen detection
- Bare-metal HAL stubs
